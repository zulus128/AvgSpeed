# AvgSpeed

Mobile application for tracking and analyzing movement speed and performance, with Apple Watch integration.

## 📱 Overview

AvgSpeed is a mobile application designed to measure, analyze, and display average speed over time.  
The app integrates with Apple Watch and provides real-time tracking, making it suitable for sports, walking, and activity monitoring.

Available on the App Store:  
[Download on the App Store](https://apps.apple.com/tr/app/avgspeed-watch/id6756965999)

[![Download on the App Store](https://developer.apple.com/assets/elements/badges/download-on-the-app-store.svg)](https://apps.apple.com/tr/app/avgspeed-watch/id6756965999)

## 🚀 Features

- Real-time speed tracking

- Apple Watch integration  
- Interactive and responsive UI

## ⌚ Apple Watch
